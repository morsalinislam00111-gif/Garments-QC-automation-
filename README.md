# Industrial QC Automation System (n8n + Apps Script)

This project is a complete automated solution designed for the Garments Industry to track production quality in real-time.

# Key Features:
* Custom Data Entry:** Built with Google Apps Script for easy and validated data input from the factory floor.
* Hourly Analytics:** Automated workflows to process line-wise and buyer-wise performance.
* Trend Analysis:** Advanced JavaScript logic to compare current data with historical trends (Up/Down indications).
* Automatic Reporting:** Generates professional PDF dashboards and delivers them via Telegram Bot instantly.

# Tech Stack:
n8n, Google Apps Script, JavaScript, Telegram Bot API, HTML-to-PDF.
